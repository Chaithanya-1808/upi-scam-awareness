# UPI Scam Awareness Website

## Overview

This project is a simple website created to spread awareness about fake UPI payment link scams. It explains how scammers trick users using phishing links and provides basic guidance on how to identify suspicious links.

## Features

* Explanation of UPI scam methods
* Examples of fake payment links
* Simple suspicious link checker
* Safety tips to prevent digital payment fraud
* Information on reporting cyber crime

## Technologies Used

* HTML
* CSS
* JavaScript

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Open **index.html** in your web browser.

## Purpose

The main goal of this project is to educate users about online payment scams and promote safe digital payment practices.
